# 心晴日记 - 情绪日记应用

基于项目文档 [emo_diary](https://github.com/MurLoper/emo_diary.git) 设计开发的完整情绪日记Web应用。

## ✨ 核心特色

### 1. 丰富的主题系统
- **8款免费主题**：粉色少女、绿意盎然、暗夜星辰、薰衣草梦境、珊瑚海滩、薄荷清新、天空之蓝、桃花粉嫩
- **3款签到解锁主题**：樱花季节(7天)、秋日枫叶(15天)、冬日雪境(30天)
- **4款积分解锁主题**：璀璨星空(300积分)、金色余晖(300积分)、克莱因蓝(300积分)、玫瑰金(500积分)
- **2款付费主题**：极光幻境(¥9.9)、银河星系(¥12.9)

### 2. 完整的签到系统
- 每日签到获得10积分
- 连续签到7天额外奖励20积分
- 连续签到30天额外奖励100积分
- 连续签到100天额外奖励500积分
- 签到日历可视化展示

### 3. 日记管理功能
- 创建、编辑、删除日记
- 上传最多9张图片
- 多标签分类系统（心情、活动、社交、天气）
- AI文字润色功能（消耗10积分）
- 搜索和筛选功能
- 15篇精心润色的示例日记

### 4. 图文集功能
- 选择多篇日记创建图文集
- 4种精美模板：经典简约、杂志风格、时间线、极简主义
- 自动提取日记标签
- 智能封面选择
- 创建图文集获得20积分

### 5. 积分激励系统
- 签到获得积分（10积分/天）
- 写日记获得积分（5积分/篇）
- 创建图文集获得积分（20积分/个）
- 积分可用于解锁主题
- 积分可用于AI润色功能

## 🎯 使用说明

### 解锁主题
1. **免费主题**：直接在主题商店点击应用即可
2. **签到主题**：
   - 连续签到达到指定天数
   - 点击主题预览
   - 点击"解锁并应用"按钮
3. **积分主题**：
   - 通过签到和写日记积累积分
   - 积分充足后点击主题
   - 使用积分解锁
4. **付费主题**：预览后可购买使用

### 获得积分
- 每日签到：10积分
- 连续签到7天：额外20积分
- 连续签到30天：额外100积分
- 连续签到100天：额外500积分
- 写日记：5积分/篇
- 创建图文集：20积分/个

### 使用积分
- AI文字润色：10积分/次
- 解锁积分主题：300-500积分
- 导出PDF：50积分（待实现）
